What to build:

A full-stack chatbot that can answer questions from:
Uploaded PDFs / documents
Web pages (user pastes URL)
SQL Database (e.g., company sales or customer data)

Features:
Conversation history + message trimming
Source citations with page numbers
Multiple retrievers (Vector + SQL)
Hybrid search
Streamlit UI + optional FastAPI backend


Tech Stack:

LangChain / LangGraph
Groq (Llama3) + Ollama fallback
ChromaDB / FAISS + AstraDB (optional)
Embeddings (OpenAI or HuggingFace)
Streamlit + optional deployment on Streamlit Cloud / Render / Hugging Face Spaces
